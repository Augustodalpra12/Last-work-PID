# algoritmo de Canny
- Verifica as regioes de maximo, armazenando os maximos e aplicando a imagem com supressão
- Usa dos limiares, sendo possível dizer o quanto de borda vc deseja preservar na figura (T1 e T2)
- É maior que T1 borda forte, maior que T2 borda fraca
- por que usa t2?: para n deixar a borda tracejada, se o cara encontrado no T2, tiver um vizinho T1 ele tambem vai participar da
borda

# método de Otsu
- Vai pegar o histograma da imagem, dividir ele e descobrir qual é o valor de cinza do limiar que maxima a variancia entre classes
  - Calcula o histograma normalizado
  - Calcula as somas acumuladas
  - Calcula as medias acumuladas
  - Calcula a média global
  - Calcula a variancia entre classes
  - Procura o valor maximo, se for encontrado mais de um maximo, o limiar otimo será a média dos valores de k que produziram
os máximos, caso contrário, escolha k como limiar.

# Algoritmo de WaterShed
- Identifica os pontos de menor intensidade na imagem (minimos locais).
- Preenche os minimos locais com "agua", a agua vai sobindo e vai se juntando a outros vales.
- A partir de quando a agua de diferentes vales se encontram, é criado barreiras para separar as regiões, representando a 
segmentação da imagem.

# Cadeia de Freeman
- Acha o primeiro ponto
- Começa achar os vizinhos em direções "horarias"
- Depois achar os vizinhos que faltaram, ate chegar no ponto inical e finalizar 
- Por ultimo pega os pontos e calcula a diferenças entre as coordenadas (anti-horaria)
e atribui os valores que tem nos dicionarios de 0 a 7.

#Filter box
- escolhe o kernel (2,3,5,7)
- passa o kernel pela imagem
- calcula a média dos pixeis dentro do kernel
- subsitui o pixel central pelo valor médio
- repeti o processo para toda a imagem
